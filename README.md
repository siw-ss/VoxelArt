## 🚀 Running Locally

This project uses vanilla HTML/CSS and requires a local web server to run.

### Using Python (Recommended)
```bash
python -m http.server 8000
```
Then open: `http://localhost:8000`

### Using Node.js (http-server)
```bash
npx http-server -p 8000
```
Then open: `http://localhost:8000`

## 🎨 Features

- **Glassmorphism Design** - Modern frosted glass UI elements
- **Responsive Grid** - Auto-fitting cards for all screen sizes
- **Interactive Hover Effects** - Smooth 3D lift animations
- **Gradient Background** - Pastel sky colors from Pagoda Garden palette
- **Emoji Favicon** - Cherry blossom icon 🌸

## 🛠️ Tech Stack

- **HTML5** - Semantic structure
- **CSS3** - Flexbox, Grid, Glassmorphism, Animations
- **Vanilla JavaScript** - No framework dependencies
